Day 13 - React fundamentals

Components, JSX, props & state

Thinking in components, JSX, props, state with useState, events, conditional rendering, lists & keys, and controlled forms.

1) Quick Start

React is a JavaScript library for building user interfaces using reusable components. We learn how to create components, pass data with props, manage changing data using state, respond to user events, conditionally render content, render lists, and organize applications into a tree of components. React updates only the parts of the page that change, making it easier to build interactive and maintainable web applications.

2) Describing the UI

We describe the user interface by building reusable React components that return JSX, a syntax that looks like HTML but is written in JavaScript. We learn how to organize components into a tree, pass data using props, and use JavaScript expressions inside JSX to create dynamic, reusable, and easy-to-maintain interfaces.

3) Adding Interactivity

We make React applications interactive by responding to user events and updating the UI when data changes. We use state to store changing data, event handlers to respond to user actions like clicks and typing, and update the interface automatically when the state changes. We also learn how React manages forms, shares state between components, and handles common interactions in a predictable way.
