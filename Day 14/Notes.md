Day 14 - Hooks, context, data fetching & hosting

Hooks in depth, the Context API & shipping Project 4

Every core hook with a real use case, the Context API for shared state, data fetching with fetch inside useEffect (loading + error states), then deploy the app.

1) Managing State

We learn how to manage and share data between React components using state. We lift state up to a common parent when multiple components need the same data, pass data down through props, and organize state so it stays consistent and easy to update. Managing state correctly helps keep the UI synchronized with the underlying data.

2) Thinking in React

We learn how to build a React application by breaking the UI into small, reusable components, creating a component hierarchy, and identifying what data should be stored as state. We decide where state should live, pass data through props, and build the app step by step, making the code easier to understand, reuse, and maintain.
